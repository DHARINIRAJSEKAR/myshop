# 🛍️ MyShop - Full Stack MERN Project

A full-stack e-commerce web application built with MERN Stack.

## 🛠️ Tech Stack
- **Frontend:** React.js, Axios, React Router
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Tools:** VS Code, MongoDB Compass

## ✨ Features

### Admin
- Dashboard with stats & revenue
- Add, Edit, Delete products
- Category & Description management
- Manage Orders — status update
- Manage Users — Activate/Deactivate

### User
- Register, Login, Forgot Password
- Search, Category filter, Sort by price
- Cart with badge count
- Wishlist
- Card Payment & Cash on Delivery
- My Orders & Cancel Order
- Profile Edit

## 🚀 How to Run

### Backend
cd myserver
npm install
npm start

### Frontend
cd myproject
npm install
npm run dev

### Open Browser
http://localhost:5173/
